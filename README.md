# homebrew-tap

Homebrew tap for [mati](https://github.com/ioni-dev/mati) — an enforcement layer
for codebase knowledge: confirmed gotchas gate what AI agents read and edit at
the hook level.

## Install

```bash
brew install ioni-dev/tap/mati
```

Or tap first, then install:

```bash
brew tap ioni-dev/tap
brew install mati
```

## Upgrade

```bash
brew update
brew upgrade mati
```

The formula is generated per release from the binaries published at
[ioni-dev/mati releases](https://github.com/ioni-dev/mati/releases).
